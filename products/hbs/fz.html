弱/非主要靶点，仅在高浓度或实验条件下观察到显著作用，临床剂量作用弱
钙调蛋白(CaM)

心血管系统分子通路
1
L型钙通道阻断
地尔硫䓬结合Cav1.2α1C亚单位→ 通道构象改变 → Ca²⁺内流↓ → 细胞内钙离子浓度([Ca²⁺]i )降低 🔬
2
心肌收缩力调节通路
[Ca²⁺]i↓→ 细胞内Ca²⁺-肌钙蛋白C结合↓→ 肌动蛋白-肌球蛋白交联↓ → 收缩力↓→ 心输出量↓ 👥
3
心率调节通路
窦房结/房室结L型钙通道阻断 → 4期自发除极速度↓/房室传导减慢 → 心率↓、PR间期延长 ；并延缓房室结传导（PR间期延长），阻滞快速房性冲动传导 👥
4
血管平滑肌舒张通路
[Ca²⁺]i↓ → 肌球蛋白轻链激酶(MLCK)活性↓→肌球蛋白去磷酸化 → 平滑肌舒张 → 外周阻力↓ 👥
5
冠状动脉扩张通路
冠脉平滑肌Ca²⁺↓ → 血管舒张 → 冠脉血流↑ → 心肌氧供↑ → 抗心绞痛作用；解除冠脉痉挛 👥

A、血流动力学机制
1
肾血管L型钙通道阻断
作用于入球小动脉Cav1.2 → 平滑肌[Ca²⁺]i↓ → 阻力↓；
拮抗Ang II介导的出球小动脉收缩（间接） 🔬
2
肾血流动力学改善
肾血管阻力↓ → 肾血流量(RBF)↑；肾小球滤过率GFR维持或轻度变化→ 肾小球高灌注/高压缓解 👥
B、非血流动力学机制
3
足细胞保护与肾小球基底膜(GBM)负电荷屏障恢复
Ca²⁺内流↓ → 足细胞凋亡↓防止足突融合；减少炎症/氧化应激对GBM硫酸乙酰肝素蛋白聚糖的损伤 → 阴离子位点密度↑、蛋白质通透率↓ → 蛋白尿下降 🔬
4
抗炎、抗氧化与抗纤维化
抑制Ca²⁺依赖NADPH氧化酶激活 → 活性氧(ROS)↓；下调TNF-α/IL-6/MCP-1等炎症因子；抑制TGF-β1/Smad → 胶原I/III沉积↓、系膜增生↓ → 延缓肾小球硬化与间质纤维化 🔬
脑血管系统分子通路
1
脑动脉 L 型钙通道阻断
作用于脑血管平滑肌Cav1.2 → 细胞内[Ca²⁺]i↓ → MLCK活性↓ → 平滑肌舒张 → 脑血管扩张 → 局部/全脑血流(CBF)↑ 🔬
2
内皮功能与NO/cGMP通路
剪切力改善 → 内皮一氧化氮合酶(eNOS)激活 → NO↑ → 环磷酸鸟苷(cGMP)↑ → 进一步血管舒张与微循环灌注改善 🔬
3
微循环与血小板活化调控
钙依赖性血小板激活倾向↓ + 微循环通透性优化 → 微血栓形成与脑水肿风险下降 🔬
4
抗缺血/再灌注损伤
减少神经元钙超载与氧自由基生成，抑制炎性因子释放与凋亡通路激活 → 脑缺血区域细胞保护（动物与转化研究为主） 🔬
